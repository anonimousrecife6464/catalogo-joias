PASSOS RÁPIDOS

1. Crie um repositório público no GitHub.
2. Envie TODO o conteúdo desta pasta para o repositório.
3. No GitHub, vá em Settings > Pages e em Build and deployment escolha GitHub Actions.
4. Aguarde o workflow terminar na aba Actions.
5. Seu site ficará online no link do GitHub Pages.

Observação: este projeto já foi ajustado com base relativa (./) e fallback SPA (404.html), o que ajuda o catálogo a funcionar no GitHub Pages.
